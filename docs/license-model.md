# License Model

Holo Rick uses a custom source-available license.

## Intent

- The public may read and study the code.
- The code is not generally open-source.
- Modification, forking, redistribution, sale, rebranding, hosted derivatives,
  and use of the UI/assets/project identity require written permission.
- Official builds must come from signed PondSec release processes.

## Why Custom

Common permissive licenses allow broad reuse and redistribution. That conflicts
with the product goal: public readability without allowing unofficial modified
commercial versions.

## Legal Review

The license is a technical project license draft and should be reviewed by a
qualified lawyer before commercial enforcement.
